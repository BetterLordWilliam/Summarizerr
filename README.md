# run the source

1. Turn on venv
```pwsh
./.venv/Scripts/Activate
```

2. Install requirements
```pwsh
pip install -r requirements.txt
```

3. Profit

4. Run the program with python3
```pwsh
python main.py
```

5. Run main with arguments
```pwsh
pythom main.py --file "/my/epic/file/path" --odir "/my/epic/directory/path"
```

**done**
